# smart-to-do-pro
Smart To Do PRO is a modern desktop task manager built with PyQt6 designed to help users organize tasks efficiently with deadlines priorities and progress tracking in a clean SaaSstyle interface. It supports task creation completion tracking scheduling with deadlines and realtime progress visualization all stored locally for fast and reliable usage
✨ Features
🧠 Add / Delete / Update tasks
📅 Deadline support for each task
🔥 Priority levels (Low / Medium / High)
📊 Live progress tracking bar
🧩 Drag & drop task reordering
💾 Local JSON storage (offline-ready)
🎨 Modern dark UI (SaaS-style design)
💻 Tech Stack
Python
PyQt6
JSON (local storage)
🎯 Goal

To provide a fast, lightweight, and beautiful desktop productivity tool that feels like a modern SaaS app but runs fully offline.
